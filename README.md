# **东南大学 Minecraft 客户端**

此仓库用以分发登录东南大学 Minecraft 服务器所需的客户端整合包，请前往 [release](https://github.com/DavidSHwang/MinecraftClientPack/releases) 页面下载整合包，并使用 [HMCL](https://hmcl.huangyuhui.net) 导入至本地。
